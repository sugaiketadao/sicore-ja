クラス情報

クラス名: FrmUtil

説明

値フォーマットクラス.
  • ページ表示する際の値のフォーマット処理を受け持つ。
  • おもにPageUtilからブラケット記法で実行され、機能単位の処理から直接実行されることはほとんどない。
  • アンフォーマット処理が対で必要。
