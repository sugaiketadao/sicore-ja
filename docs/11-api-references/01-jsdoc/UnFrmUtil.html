クラス情報

クラス名: UnFrmUtil

説明

値アンフォーマットクラス.
  • リクエスト作成に際して値のアンフォーマット処理を受け持つ。
  • おもにPageUtilからブラケット記法で実行され、機能単位の処理から直接実行されることはほとんどない。
  • フォーマット処理が対で必要。
