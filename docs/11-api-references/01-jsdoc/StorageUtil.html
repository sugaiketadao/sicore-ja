クラス情報

クラス名: StorageUtil

説明

セッションストレージユーティリティクラス.
  • ブラウザのセッションストレージに下記の単位＋キーで連想配列を格納・取得する。
    • ページ単位（URLの HTMLファイル単位、１ページ内でデータ保持）
    • 機能単位（URLの機能ディレクトリ単位、ページ間でデータ共有）
    • システム単位（システム全体でデータ共有）
  • クリティカルな処理ではない前提とし、原則として例外エラーとしない。
