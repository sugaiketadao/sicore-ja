メソッド一覧

convUrlParam

convUrlParam(obj)
連想配列 to URLパラメーター変換.
  • ［例］
     params = {p1: 'aaa', p2: 'bbb'} の場合
     p1=aaa&p2=bbb に変換する。

パラメータ

名前 型 説明
obj Object.<string,string> 連想配列

戻り値

string - URLパラメーター文字列

getUrlParams

getUrlParams()
URLパラメーター取得.
  • URLの?以降を連想配列で取得する。
  • ［例］ 「a=01&b=02」の場合、{a:'01', b:'02'} を返す。
  • パラメーターは取得後に削除される。

戻り値

Object - URLパラメーター連想配列

movePage

movePage(url, params)
ページ遷移.
  • 指定URLに遷移する。（HTMLファイル取得）
  • パラメーターを指定した場合はURLの?以降に付与する。
  • ［例］
     url = 'editpage.html'、params = {user_id: 'U001', upd_ts: '20251231T245959001000'} の場合
     editpage.html?user_id=U001&upd_ts=20251231T245959001000 にアクセスする。

パラメータ

名前 型 説明
url string 遷移先URL
[params] Object.<string, string>|string パラメーター（文字列も可能）（省略可能）
