クラス情報

クラス名: SessionUtil

説明

セッションユーティリティクラス.
  • Webサービス呼び出し時にリクエストとともに送信され、レスポンスデータで自動更新される。
  • JavaScript の変数（インメモリ）でデータを保持する。
  • リンクでのページ遷移やリロードでデータが失われる。
  • ページ遷移は HttpUtil.movePage を使用することでデータを保持できる。
  • JWT を保持し Webサービス呼び出し時、リクエストヘッダーに自動的に付与される。
