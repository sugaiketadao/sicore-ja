isTrue

isTrue(val)
真偽値チェック.
  • 文字列が真偽値「真」とみなす値かチェックする。
  • 下記の評価をおこなう。
    1. "1", "true", "yes", "on"（すべて半角）は true。
    2. null またはブランクは false を含み、上記以外は false。
    3. 大文字小文字を区別しない。
    4. 左右の半角ブランクは無視する。
    5. boolean値はそのまま返す。

パラメータ

名前 型 説明
val string|boolean チェック対象

戻り値

boolean - 真偽値「真」とみなす場合は true
