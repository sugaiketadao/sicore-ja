クラス名: BreakException

継承: RuntimeException

説明

ブレーク例外クラス.
  • 処理を中止するために使用する例外クラスです。
  • 本クラスをスローするとスタックトレースの出力が中断されます。（LogUtil.getStackTrace() メソッドで考慮されています。）
  • 既にログ出力しており、エラーの伝播先で再度トレースログを出力されたくない場合に使用します。
